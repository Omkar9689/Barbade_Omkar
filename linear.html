Linear Search

What is Linear Search?

Linear search, also known as sequential search, is a method for finding a target value within a list. It sequentially checks each element of the list until a match is found or the whole list has been searched.

Example:

Consider the following array: [10, 20, 30, 40, 50, 60]

Searching for the value 40:

Output:


        40 found at index 3
      

    


    

      
Complexity Analysis of Linear Search:

      
Time Complexity: O(N)

      
Auxiliary Space: O(1)

    


    

      
Advantages of Linear Search:

      
    
        
  • Simple and easy to implement.
    • 
        
  • Works well for small lists or when the target value is near the beginning of the list.
    • 
        
  • Does not require the list to be sorted.
    • 
      

    


    

      
Disadvantages of Linear Search:

      
    
        
  • Linear search has a time complexity of O(N), which makes it inefficient for large lists.
    • 
        
  • Not suitable for situations where the target value is near the end of the list or not present in the list.
