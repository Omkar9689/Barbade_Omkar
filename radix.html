Radix Sort

What is Radix Sort?

Radix Sort is a non-comparison-based sorting algorithm that sorts integers by processing individual digits of numbers. It sorts numbers by first grouping the elements by their individual digits, then sorting the elements according to their digit values.

Example:

Consider the following array: [170, 45, 75, 90, 802, 24, 2, 66]

Output:


        

          Applying Radix Sort:
          [2, 24, 45, 66, 75, 90, 170, 802]
        

      

    


    

      
Complexity Analysis of Radix Sort:

      
Time Complexity: O(N * k)

      
Auxiliary Space: O(N + k)

      
Where N is the number of elements in the array and k is the number of digits in the maximum element.

    


    

      
Advantages of Radix Sort:

      
    
        
  • Linear time complexity for sorting integers.
    • 
        
  • Efficient for sorting numbers with a fixed number of digits.
    • 
        
  • Doesn't require comparison operations.
    • 
        
  • Stable; maintains the relative order of elements with equal keys.
    • 
      

    


    

      
Disadvantages of Radix Sort:

      
    
        
  • Requires additional space proportional to the size of the input array.
    • 
        
  • Only suitable for sorting integer keys.
    • 
        
  • Not as efficient for sorting floating-point numbers or variable-length keys.
