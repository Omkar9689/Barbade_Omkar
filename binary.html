Binary Search

What is Binary Search?

Binary search is a fast search algorithm with a time complexity of O(log n). This search algorithm works on the principle of divide and conquer. It works only on a sorted array, so the given array must be sorted before applying binary search.

Example:

Consider the following sorted array: [10, 20, 30, 40, 50, 60]

Searching for the value 30:

Output:


        30 found at index 2
      

    


    

      
Complexity Analysis of Binary Search:

      
Time Complexity: O(log n)

      
Auxiliary Space: O(1)

    


    

      
Advantages of Binary Search:

      
    
        
  • Efficient for searching in large sorted arrays.
    • 
        
  • Has a time complexity of O(log n).
    • 
      

    


    

      
Disadvantages of Binary Search:

      
    
        
  • Requires the array to be sorted before applying binary search.
    • 
        
  • Not suitable for unsorted arrays or dynamic data structures like linked lists.
