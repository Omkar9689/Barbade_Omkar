Probability Search

What is Probability Search?

Probability search, also known as Interpolation Search, is an efficient search algorithm used for searching in sorted arrays with uniformly distributed values.

How does it work?

Unlike binary search which always checks the middle element, probability search calculates the probable position of the target element using interpolation formula and adjusts the search position accordingly.

Time Complexity:

Best Case: O(1)

Average Case: O(log log n)

Worst Case: O(n)

Advantages of Probability Search:

  • Efficient for large datasets with uniformly distributed values.
  • Can perform better than binary search in certain scenarios.

Disadvantages of Probability Search:

  • Requires the array to be sorted and uniformly distributed.
  • Not suitable for small datasets or datasets with non-uniform distributions.
