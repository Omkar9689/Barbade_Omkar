Insertion Sort

What is Insertion Sort?

Insertion Sort is a simple sorting algorithm that builds the final sorted array (or list) one item at a time. It is much less efficient on large lists than more advanced algorithms such as quicksort, heapsort, or merge sort.

Example:

Consider the following array: [12, 11, 13, 5, 6]

Output:


        

          Applying Insertion Sort:
          [5, 6, 11, 12, 13]
        

      

    


    

      
Complexity Analysis of Insertion Sort:

      
Time Complexity: O(N2)

      
Auxiliary Space: O(1)

    


    

      
Advantages of Insertion Sort:

      
    
        
  • Simple implementation.
    • 
        
  • Efficient for small data sets.
    • 
        
  • Adaptive, efficient if the input is already substantially sorted.
    • 
        
  • Stable; does not change the relative order of elements with equal keys.
    • 
      

    


    

      
Disadvantages of Insertion Sort:

      
    
        
  • More complex algorithms (such as quicksort, heapsort, or merge sort) are more efficient for larger lists.
    • 
        
  • Time complexity is O(N2), making it inefficient for large lists.
    • 
        
  • Not suitable for large data sets.
