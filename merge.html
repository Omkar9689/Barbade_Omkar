Merge Sort

What is Merge Sort?

Merge Sort is a popular sorting algorithm that follows the Divide and Conquer strategy. It divides the input array into two halves, sorts each half independently, and then merges the sorted halves to produce a single sorted array.

Example:

Consider the following array: [38, 27, 43, 3, 9, 82, 10]

Output:


        

          Applying Merge Sort:
          [3, 9, 10, 27, 38, 43, 82]
        

      

    


    

      
Complexity Analysis of Merge Sort:

      
Time Complexity: O(N log N)

      
Auxiliary Space: O(N)

    


    

      
Advantages of Merge Sort:

      
    
        
  • Guaranteed O(N log N) time complexity.
    • 
        
  • Efficient for large datasets.
    • 
        
  • Stable; does not change the relative order of equal elements.
    • 
        
  • Good for sorting linked lists as well as arrays.
    • 
      

    


    

      
Disadvantages of Merge Sort:

      
    
        
  • Requires additional space proportional to the input size.
    • 
        
  • Not as efficient for small datasets or nearly sorted arrays as other algorithms like insertion sort.
    • 
        
  • Recursive implementation may cause stack overflow for very large datasets.
