Sentinel Search

What is Sentinel Search?

Sentinel search, also known as sentinel linear search, is a variant of linear search. It improves the performance of linear search by reducing the number of comparisons.

Example:

Consider the following array: [10, 20, 30, 40, 50, 60]

Searching for the value 30:

Output:


        50 found at index 4
      

    


    

      
Advantages of Sentinel Search:

      
    
        
  • Reduces the number of comparisons in linear search.
    • 
        
  • Simple and easy to implement.
    • 
      

    


    

      
Disadvantages of Sentinel Search:

      
    
        
  • Requires an additional element (sentinel) in the array.
    • 
        
  • Not suitable for situations where memory usage is a concern.
