Order Search

What is Order Search?

Order search, also known as Sequential Search, is a simple search algorithm that sequentially checks each element of the array until a match is found or the entire array has been traversed.

How does it work?

Order search iterates through each element of the array, comparing it with the target value. If the element matches the target, the search stops and returns the index of the element.

Time Complexity:

Best Case: O(1)

Average Case: O(n/2)

Worst Case: O(n)

Advantages of Order Search:

  • Simple and easy to implement.
  • Works well for small datasets or unsorted arrays.

Disadvantages of Order Search:

  • Linear time complexity makes it inefficient for large datasets.
  • Not suitable for sorted arrays where binary search can be applied.
